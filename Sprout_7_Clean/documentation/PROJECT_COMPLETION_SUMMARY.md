## 🚀 **HOW TO USE (FINAL INSTRUCTIONS)**

### **🆕 For New Computers:**
1. **Windows**: Double-click `SETUP_NEW_COMPUTER.bat`
2. **Linux/macOS**: Run `chmod +x setup_new_computer.sh && ./setup_new_computer.sh`
3. **Wait 10-30 minutes** for complete automated setup
4. **Follow on-screen instructions** for any manual steps needed

### **▶️ To Start the System:**
1. **Windows**: Run `START_QWEN_ENHANCED.bat`
2. **Linux/macOS**: Run `./start_krishi_sakhi.sh` (created by setup)
3. **Open browser**: Go to http://localhost:8000
4. **Start chatting**: AI is always visible at the top!
