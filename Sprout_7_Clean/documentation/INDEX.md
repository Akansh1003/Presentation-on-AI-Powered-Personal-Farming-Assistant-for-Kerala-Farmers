## 🚀 **Quick Setup Guide**

### **🆕 NEW COMPUTER SETUP**

**Windows Users:**
```batch
# Double-click to run:
SETUP_NEW_COMPUTER.bat
```

**Linux/macOS Users:**
```bash
chmod +x setup_new_computer.sh
./setup_new_computer.sh
```

### **▶️ START THE SYSTEM**

**Windows:**
```batch
START_QWEN_ENHANCED.bat
```

**Linux/macOS:**
```bash
./start_krishi_sakhi.sh  # Created by setup script
```

### **🌐 ACCESS THE APPLICATION**
- **Main Interface**: http://localhost:8000
- **Backend API**: http://localhost:5000
