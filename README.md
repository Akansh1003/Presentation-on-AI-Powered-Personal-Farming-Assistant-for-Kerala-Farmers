## 🌟 **What is Krishi Sakhi?**

**Krishi Sakhi** (कृषि सखी - meaning "Farming Friend" in Hindi) is a revolutionary AI-powered farming assistant specifically designed for Kerala's unique agricultural landscape. Born from the need to democratize agricultural knowledge, this system combines cutting-edge artificial intelligence with deep domain expertise to provide Kerala's farmers with instant, personalized, and scientifically accurate farming guidance.

### **The Vision**
To bridge the knowledge gap between traditional farming wisdom and modern agricultural science, making expert-level farming advice accessible to every farmer in Kerala, regardless of their location or economic background.

---

## 🎯 **Why Krishi Sakhi?**

### **The Problem We Solve**
- **Knowledge Gap**: Many farmers lack access to up-to-date agricultural information
- **Language Barriers**: Most farming resources are not available in local languages
- **Timely Advice**: Farmers need immediate answers for time-sensitive agricultural decisions
- **Personalization**: Generic advice doesn't account for local soil, climate, and crop conditions
- **Cost**: Hiring agricultural consultants is expensive for small-scale farmers

### **Our Solution**
- 🤖 **24/7 AI Assistant** available in Malayalam and English
- 🌾 **Kerala-Specific Knowledge** tailored to local crops, climate, and conditions
- 📱 **Easy-to-Use Interface** accessible via any web browser
- 🆓 **Completely Free** - no hidden costs or subscriptions
- 🧠 **Continuous Learning** - gets smarter with every interaction

## 🛠️ **How It Works**

### **Simple 3-Step Process**
1. **Ask Your Question** - Type in Malayalam or English, or click feature cards
2. **AI Processing** - Advanced models analyze your query and fetch relevant data
3. **Get Expert Answer** - Receive detailed, personalized farming advice instantly

### **Behind the Scenes**
```
User Input → Frontend Processing → API Gateway → AI Analysis → Knowledge Base → Response Generation → User Display
```

The system intelligently routes questions through multiple AI models and knowledge bases to ensure the most accurate and helpful response possible.

## 🛠️ **Installation & Setup**

### **🚀 NEW COMPUTER SETUP (Recommended)**

**For completely new computers or first-time installations:**

#### **Windows:**
```batch
# Run the automated setup script
SETUP_NEW_COMPUTER.bat
```

#### **Linux/macOS/Unix:**
```bash
# Make the script executable and run
chmod +x setup_new_computer.sh
./setup_new_computer.sh
```

**These scripts will:**
- ✅ Check and install Python 3.8+ automatically
- ✅ Install all required dependencies and packages
- ✅ Create necessary directory structure
- ✅ Initialize SQLite database with proper schema
- ✅ Download AI models on first run
- ✅ Verify everything is working correctly
- ✅ Provide detailed setup status and next steps

---

### **🔧 Manual Setup (Advanced Users)**

**If you prefer manual installation or have existing Python setup:**

```bash
# 1. Clone or download the project
git clone <repository-url>
cd Sprout_7_Clean

# 2. Install Python dependencies
pip install -r requirements.txt
# OR install individually:
pip install flask flask-cors flask-sqlalchemy transformers torch requests

# 3. Start the system
# Windows:
START_QWEN_ENHANCED.bat

# Linux/macOS:
./start_krishi_sakhi.sh
```

### **📦 Requirements File**
We provide a comprehensive `requirements.txt` file with all dependencies:
```bash
pip install -r requirements.txt
```

### **🎯 After Setup**
Once setup is complete:
1. **Access the application**: http://localhost:8000
2. **Chat interface**: Always visible at the top of the page
3. **Register**: Create your farmer profile for personalized advice
4. **Start chatting**: Ask farming questions in English or Malayalam
